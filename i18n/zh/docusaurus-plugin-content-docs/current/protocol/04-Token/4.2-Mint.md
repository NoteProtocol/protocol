## 4.2 分发 Mint

Mint按照下面的数据定义。

```json
{ 
  "p": "n20",
  "op": "mint",
  "tick": "note",
  "amt": "1000"
  ... ...
}
```

| 名称 | 是否必须 | 描述 |
| :--- | :--- | --- |
| p | 必须 | 协议名称 n20 小写 |
| op | 必须 | 操作名称 mint 小写 |
| tick | 必须 | 代币名称 最大16个字节 大小写不敏感 |
| amt | 必须 | Bigint 大整数, 本次挖掘的数量, 大于0 |

如在deploy阶段设置了智能合约，Mint必须能通过合约验证。
